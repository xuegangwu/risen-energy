# 工商业储能团购平台 - 项目规格文档

## 1. 项目概述

**项目名称：** 储能智选商城
**项目类型：** B2B 电商平台
**核心功能：** 工商业储能系统标准化套餐展示、在线选购、团购、订单管理
**目标用户：** 工商业终端用户（工厂、园区、商场）、渠道商/经销商

## 2. 功能模块

### 2.1 套餐系统
- **标准套餐展示**：电池系统、变流器、EMS、配电设备、光伏配套
- **套餐详情**：配置参数、价格、适用范围
- **套餐分类**：按容量、按应用场景

### 2.2 购物车与订单
- **在线选购**：加入购物车、修改数量
- **团购功能**：凑单人数阶梯价格
- **订单管理**：订单创建、支付、状态跟踪

### 2.3 用户系统
- **用户注册/登录**：手机号、邮箱
- **企业认证**：营业执照上传
- **个人中心**：订单 history、收藏、消息

### 2.4 支付系统
- **在线支付**：微信支付、支付宝
- **线下支付**：银行转账
- **发票管理**

### 2.5 团购系统
- **拼团活动**：限时拼团、阶梯价格
- **团购进度**：实时显示已参团人数

## 3. 技术架构

### 后端
- **框架**：Node.js + Express
- **数据库**：MongoDB
- **认证**：JWT
- **支付集成**：微信支付、支付宝

### 前端
- **框架**：React + 
- **路由**：React Router
- **状态管理**：Zustand
- **UI组件**：Tailwind CSS + Radix UI
- **HTTP客户端**：Axios

### 部署
- **服务器**：国内云服务器（推荐阿里云/腾讯云）
- **反向代理**：Nginx
- **域名**：已有域名

## 4. 数据库设计

### 用户集合 (users)
- 基础信息（用户名、密码、手机、邮箱）
- 企业信息（公司名、营业执照）
- 认证状态

### 套餐集合 (packages)
- 套餐名称、分类、价格
- 组件配置（电池、变流器、EMS等）
- 图片、描述

### 订单集合 (orders)
- 用户ID、套餐信息
- 订单金额、支付状态
- 物流信息

### 团购集合 (groups)
- 套餐ID、拼团价格
- 成团人数、当前人数
- 截止时间

## 5. 页面结构

1. **首页** - 热门套餐、活动展示
2. **套餐列表** - 分类浏览、筛选
3. **套餐详情** - 配置参数、加入购物车
4. **购物车** - 商品列表、结算
5. **订单确认** - 地址、支付方式
6. **用户中心** - 订单 history、个人信息
7. **团购专区** - 进行中的拼团活动
8. **关于我们** - 公司介绍、联系方式
